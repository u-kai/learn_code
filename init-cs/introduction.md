# プログラミング言語ってなに？

人がコンピュータにやって欲しいことを記述するのに適している記述方式

- いっぱい種類がある
  - c/c++
  - python
  - javascript/typescript
  - java
  - rust
  - ...
- 各々に得意分野があるので，やりたいことに合わせて言語を選ぶことが大事
- 基本的には一枚のファイルには一つの言語

---

# なんでプログラミング言語って必要なの？

コンピュータは 0 と 1 しかわからないから
コンピュータは電源の ON を 1OFF を 0 と認識している
